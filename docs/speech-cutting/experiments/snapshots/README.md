# 语音切分实验证据快照

本目录从被 `.gitignore` 排除的 `outputs/` 中保存了三轮实验的关键小型证据：

- `alignment-benchmark/`：24 边界的四路 provider 输出和汇总报告。
- `first-blind-test/`：第一轮 30 秒盲听的切点清单、QC 和盲键。
- `sentence-deletion/`：最终整句跳剪的切点清单、QC、放大镜清单和盲键。

这些文件是 2026-07-13 整理时的原始快照，不应手工修改数字。部分 JSON 保留了当时本机绝对路径，路径只是运行环境记录，不影响结果解读。完整音频仍在 `outputs/`，没有复制进 Git。
